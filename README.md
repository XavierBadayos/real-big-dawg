# Stat Muncher

NBA stats filter and sort. Now includes true shooting, usage percentage, and pace from advanced stats.


## Getting Started

1. Clone the repository to your machine.
2. Open a terminal in the project's root file.
3. Install dependencies ``pnpm install``
4. Start the development server ``pnpm dev``
5. Navigate in your web browser to the local port shown in the terminal.

##Features

1. Filter stats
2. Sort stats
